1. 화면 한눈에 보기

화면은 크게 3구역입니다. 왼쪽에서 입력하면 오른쪽 미리보기가 실시간으로 그려집니다. (별도 "만들기" 버튼 없음)

구역설명
① 입력 패널내용·디자인을 입력. 위쪽 두 탭 디자인 설정 항목 구성 으로 나뉨
② 미리보기입력 즉시 완성 이미지가 표시 (0.3초 뒤 자동 갱신)
③ 리모컨오른쪽 아래 동그란 버튼들 — 다운로드·백업·되돌리기

2. 가장 빠른 사용 흐름

  1. 항목 구성 탭에서 제목 항목 버튼으로 이벤트 이름·가격을 입력
  2. 디자인 설정 탭에서 제목·기간·색상을 입력
  3. 오른쪽 미리보기로 결과 확인
  4. 오른쪽 아래 파란 다운로드 버튼으로 이미지 저장
  5. 빨간 백업 저장 버튼으로 작업 내용을 파일로 보관

3. 「항목 구성」 탭 — 내용 입력

전단지에 들어갈 이벤트 목록·가격을 만드는 곳입니다.

맨 아래 추가 버튼 3개

버튼하는 일
제목새 섹션(이벤트 묶음) 제목 추가 (예: "보톡스 이벤트")
항목이벤트 한 줄 추가 (이름 + 가격)
컬럼좌·우 2단으로 나누는 구분선 추가

항목·제목 한 줄 구성

  • 항목명 칸 — 이벤트 이름 (예: 아쿠아필(블랙헤드))
  • 단위 / 금액 칸 — 가격 (예: 단위 "5회", 금액 "20만원"). 옆 ＋로 가격 여러 개 추가
  • 소제목 — 가격 없이 강조만 하는 줄로 전환(소분류 제목용)
  • 비고 — 아래 작은 글씨 메모 칸 추가 (예: ※ 부가세 별도)
  • 복제 / 삭제 · 왼쪽 점 6개(⠿)를 끌어 순서 이동
  • 제목(섹션) 옵션(막대 아이콘): 제목/본문/금액 크기 ± 조절, 레이아웃 2열 / 전체폭

위쪽 보조 기능

  • 컬럼 필터 — 좌·우 중 한쪽만 골라 보며 편집
  • 모두 접기 / 모두 펼치기 — 모든 섹션을 한 번에 접거나 펼침

4. 글자 꾸미기 (서식 버튼)

항목명·제목 칸 아래 작은 버튼으로 글자를 꾸밉니다. 꾸밀 글자를 마우스로 드래그해 선택한 뒤 버튼을 누르세요.

버튼효과
강조선택 글자에 강조색(기본 노란 형광) 적용
확대 / 축소글자를 한 단계 크게 / 작게
굵게글자를 두껍게 (항목명에서만)
💡 글자를 선택하지 않고 누르면 빈 표시만 들어갑니다. 먼저 글자를 선택하세요.

5. 「디자인 설정」 탭 — 꾸미기

배경 이미지

클릭하거나 사진을 끌어다 놓으면 배경으로 깔립니다. (선택사항, 20MB 이하)

색상 5종

색상어디에 쓰이나
테마 색상전체 기본 색(제목 등)
숫자 색상가격 숫자 색
상단 텍스트 색상맨 위 제목·기간 글자 색
강조 색상강조 버튼으로 칠하는 색
단위박스 색상가격 옆 단위 칩(작은 박스) 배경색

색칸 클릭(색 선택) 또는 #000000 칸에 코드 직접 입력. 자주 쓰는 색은 ＋(커스텀 저장)으로 최대 6개 보관.

글자 & 크기

  • 제목 / 이벤트 기간 / 기간 우측 텍스트 (서식 버튼 사용 가능)
  • 헤더 높이 — 맨 위 제목 영역 높이(기본 5%). 0이면 제목·기간이 안 나옵니다.
  • 전체 텍스트 크기 — 전체 글자 배율(70~150%)

6. ⭐ 자동 맞춤 기능 (핵심)

전단지를 A4 한 장에 깔끔히 채워주는 자동 기능입니다.

① 「한 장에 자동 맞춤」 기본 켜짐 · 권장

  • 내용이 항상 A4 한 장에 들어가도록 글자 크기를 자동 조절합니다.
  • 켜져 있으면 전체 텍스트 크기 슬라이더는 잠금(자동 제어)됩니다.
  • 너무 많아 최소 크기로도 안 들어가면 → "항목 수를 줄이세요" 안내가 뜹니다.
  • 직접 크기를 정하려면 체크를 끄면 슬라이더가 풀립니다.

② 「좌우 균등 맞춤」 버튼

  • 섹션들을 좌·우 두 칸에 높이가 비슷하게 자동 분배합니다.
  • 켠 뒤 내용을 바꾸면 버튼이 주황색 「다시 맞추기 ⟳」로 바뀝니다 → 누르면 다시 정렬.
  • 한 번 더 누르면 원래 배치로 되돌아갑니다.

③ 「상하단 정확 맞춤」 (균등 맞춤의 세부 옵션)

  • 켜짐: 좌·우 두 칸의 아래 끝을 페이지 끝에 딱 맞춤
  • 꺼짐: 간격을 좁게 유지(짧은 쪽 아래에 여백 생길 수 있음)

7. 오른쪽 미리보기

입력하는 즉시 완성 이미지가 표시됩니다. 위쪽 상태 표시:

  • N페이지 완료 — 정상 완성
  • ⚠ 경고 — 내용이 넘쳐 잘렸거나, 헤더 높이가 0이라 제목이 안 보임 등

9. 저장과 백업 (중요!)

  • 작업 내용은 브라우저에 자동 임시 저장되지만, 탭(창)을 닫으면 사라질 수 있습니다.
  • 중요한 작업은 꼭 「백업 저장」(빨간 버튼)으로 파일을 받아 두세요. (예: A4이벤트_백업_20260601_1430.json)
  • 다음에 이어서 작업할 땐 「백업 불러오기」로 그 파일을 엽니다.
✅ 습관 추천: 완성하면 ① 이미지 다운로드(인쇄용 JPG) + ② 백업 저장(.json) 둘 다 받아 두기.

10. 자동 안전장치

  • 의료법 금지어 감지 — 항목명에 광고 금지어(예: "신데렐라주사")를 입력하면 경고와 대체어(예: "티옥트산주사")를 알려줍니다.
  • 넘침 경고 — 내용이 한 장을 넘으면 미리보기 위에 경고가 떠 알려줍니다.

11. 자주 묻는 질문

질문답
만들기 버튼이 없어요입력하면 자동으로 그려집니다. 별도 버튼이 없는 게 정상입니다.
글자가 갑자기 커지거나 작아져요한 장에 자동 맞춤이 켜져 한 장에 맞추는 중입니다. 직접 정하려면 체크를 끄세요.
작업하던 게 사라졌어요탭을 닫으면 임시 저장이 지워질 수 있습니다. 항상 백업 저장으로 파일을 받아 두세요.
글자가 잘려요 / "다 안 들어간다"고 떠요항목이 너무 많습니다. 항목을 줄이거나 컬럼으로 좌우 2단으로 나누세요.
인터넷이 안 되면?이미지 만들기·1장 다운로드는 됩니다. 단, 글꼴 모양·백업 저장은 인터넷이 필요할 수 있습니다.
