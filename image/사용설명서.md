# 원내이미지 생성기 — 사용 설명서

병원 이벤트 **A4 전단지 이미지**를 마우스 클릭만으로 만드는 도구입니다.
설치 필요 없이 **인터넷 주소만 열면** 바로 사용합니다.

> 접속 주소: **https://younnge.github.io/barog-image-generator/**

---

## 1. 화면 한눈에 보기

화면은 크게 **3구역**입니다.

```
┌──────────┬──────────────────────┬───────────────┐
│ 북마크    │   ① 입력 패널(왼쪽)    │ ② 미리보기(오른쪽) │
│ (목차)    │  - 디자인 설정 탭      │  완성 이미지가     │
│          │  - 항목 구성 탭        │  실시간 표시      │
└──────────┴──────────────────────┴───────────────┘
                                    ③ 오른쪽 아래 동그란 리모컨
                                    (저장·다운로드·되돌리기)
```

- **① 입력 패널** — 내용과 디자인을 입력하는 곳. 위쪽 두 개 탭(`디자인 설정` / `항목 구성`)으로 나뉩니다.
- **② 미리보기** — 입력하는 즉시 완성 이미지가 오른쪽에 나타납니다. (별도 "만들기" 버튼 없음)
- **③ 리모컨** — 화면 오른쪽 아래에 떠 있는 동그란 버튼들. 다운로드·백업 등.

> 💡 **모든 변경은 실시간 반영됩니다.** 입력하면 0.3초 뒤 미리보기가 자동으로 새로 그려집니다.

---

## 2. 가장 빠른 사용 흐름 (5단계)

1. **`항목 구성` 탭**에서 `제목`과 `항목` 버튼으로 내용(이벤트 이름·가격)을 입력합니다.
2. **`디자인 설정` 탭**에서 제목·기간·색상을 입력합니다.
3. **오른쪽 미리보기**로 결과를 확인합니다.
4. 마음에 들면 오른쪽 아래 **파란 다운로드 버튼**으로 이미지를 저장합니다.
5. 나중에 또 쓰려면 **빨간 백업 저장 버튼**으로 작업 내용을 파일로 보관합니다.

---

## 3. 「항목 구성」 탭 — 내용 입력

전단지에 들어갈 **이벤트 목록·가격**을 만드는 곳입니다.

### 3-1. 맨 아래 추가 버튼
| 버튼 | 하는 일 |
|---|---|
| **제목** | 새 섹션(이벤트 묶음) 제목을 추가 (예: "보톡스 이벤트") |
| **항목** | 이벤트 한 줄 추가 (이름 + 가격) |

> 💡 좌/우 2단으로 나눌 때는 위쪽 **`우측 이벤트` 탭**을 누르면 됩니다(자동으로 구분선 생성). → 3-4 참고

### 3-2. 「항목」 한 줄 구성
항목을 추가하면 한 줄에 다음이 있습니다.

- **항목명 칸** — 이벤트 이름 입력 (예: 아쿠아필(블랙헤드))
- **단위 / 금액 칸** — 가격 입력 (예: 단위 "1회", 금액 "20만원")
  - 칸 옆 **＋** : 가격을 여러 개로 늘리기 (예: 1회 / 3회 / 5회)
- **소제목** 버튼 — 가격 없이 강조만 하는 줄로 바꿈 (소분류 제목용)
- **비고** 버튼 — 아래에 작은 글씨 메모 칸 추가 (예: ※ 부가세 별도)
- **복제 / 삭제** — 줄 복사 / 줄 지우기
- **왼쪽 점 6개(⠿)** — 마우스로 끌어 **순서 이동**

### 3-3. 「제목」(섹션) 구성
- **제목 칸** — 섹션 이름 입력
- **옵션 버튼**(막대 아이콘)을 누르면 펼쳐지는 항목:
  - **제목 크기 / 본문 크기 / 금액 크기** — `−` `＋`로 글자 크기 조절
  - **레이아웃** — `2열`(좌우 나눠 배치) / `전체폭`(가로 전체 사용) 선택
- **복제 / 삭제**
- **섹션 앞쪽**을 누르면 그 섹션을 **접기/펼치기**

### 3-4. 위쪽 보조 기능
- **컬럼 탭**(`좌측 이벤트` / `우측 이벤트`) — 전단지를 좌·우 2단으로 나눠 편집
  - 기본은 **`좌측 이벤트`** 가 선택돼 있고, 컬럼을 나누기 전에는 **모든 항목이 좌측 탭에 보입니다.**
  - **`우측 이벤트` 탭을 누르면 자동으로 좌·우 구분선이 생겨**, 그 뒤에 추가하는 항목이 우측 칸에 들어갑니다.
  - 각 탭은 해당 칸 항목만 골라 보여 줘 편집이 편합니다.
- **모두 접기 / 모두 펼치기** — 모든 섹션을 한 번에 접거나 펼침

---

## 4. 글자 꾸미기 (서식 버튼)

항목명·제목 칸 **아래에 있는 작은 버튼**으로 글자를 꾸밉니다.
**꾸밀 글자를 마우스로 드래그해 선택한 뒤** 버튼을 누르면 됩니다.

| 버튼 | 효과 |
|---|---|
| **강조** | 선택 글자에 강조색(기본 노란 형광) 적용 |
| **확대** | 글자를 한 단계 크게 |
| **축소** | 글자를 한 단계 작게 |
| **굵게** | 글자를 두껍게 (항목명에서만) |

> 💡 글자를 선택하지 않고 누르면 빈 표시만 들어가니, **먼저 글자를 선택**하세요.

---

## 5. 「디자인 설정」 탭 — 꾸미기

전체 디자인(색·제목·크기)을 정하는 곳입니다.

### 5-1. 배경 이미지
- **클릭하거나 사진을 끌어다 놓으면** 배경으로 깔립니다. (선택사항, 20MB 이하)

### 5-2. 색상 (5종)
| 색상 | 어디에 쓰이나 |
|---|---|
| **테마 색상** | 전체 기본 색(제목 등) |
| **숫자 색상** | 가격 숫자 색 |
| **상단 텍스트 색상** | 맨 위 제목·기간 글자 색 |
| **강조 색상** | `강조` 버튼으로 칠하는 색 |
| **단위박스 색상** | 가격 옆 단위 칩(작은 박스) 배경색 |

각 색상은 ① **네모 색칸 클릭 → 색 선택**, 또는 ② **`#000000` 칸에 색상코드 직접 입력**.
자주 쓰는 색은 **＋(커스텀 저장)** 으로 최대 6개까지 저장해 둘 수 있습니다.

### 5-3. 글자 입력
- **제목** — 전단지 맨 위 큰 제목 (예: 바로그 강남점 5월 이벤트)
- **이벤트 기간** — 날짜 (예: 2026.05.01 - 2026.05.31)
- **기간 우측 텍스트** — 기간 옆 작은 메모 (예: 부가세 별도)
- 이 칸들도 **확대/축소/굵게** 서식 버튼이 있습니다.

### 5-4. 크기 슬라이더 2개
- **헤더 높이** — 맨 위 제목 영역의 높이 (기본 5%). `0`으로 하면 제목·기간이 안 나옵니다.
- **전체 텍스트 크기** — 전체 글자 배율 (70~150%).

---

## 6. ⭐ 자동 맞춤 기능 (핵심)

전단지를 **A4 한 장에 깔끔히** 채워주는 자동 기능입니다.

### 6-1. 「한 장에 자동 맞춤」 (기본 켜짐) ✅ 권장
- 내용이 **항상 A4 한 장에 들어가도록 글자 크기를 자동 조절**합니다.
- 켜져 있으면 `전체 텍스트 크기` 슬라이더는 **자동 제어(잠금)** 됩니다.
- 내용이 너무 많아 최소 크기로도 안 들어가면 → **"항목 수를 줄이세요"** 안내가 뜹니다.
- 직접 크기를 조절하고 싶으면 **체크를 끄면** 슬라이더가 풀립니다.

### 6-2. 「좌우 균등 맞춤」 버튼
- 누르면 섹션들을 **좌·우 두 칸에 높이가 비슷하게 자동 분배**합니다.
- 켜진 뒤 내용을 바꾸면 버튼이 주황색 **`다시 맞추기 ⟳`** 로 변합니다 → 누르면 다시 정렬됩니다.
- 한 번 더 누르면 **원래 배치로 되돌아갑니다.**

### 6-3. 「상하단 정확 맞춤」 (균등 맞춤의 세부 옵션)
- **켜짐**: 좌·우 두 칸의 **아래 끝을 페이지 끝에 딱 맞춤**.
- **꺼짐**: 간격을 좁게 유지(짧은 쪽 아래에 여백 생길 수 있음).

---

## 7. 오른쪽 미리보기

- 입력하는 즉시 완성 이미지가 표시됩니다.
- 위쪽 **상태 표시**:
  - **N페이지 완료** — 정상 완성
  - **⚠ 경고** — 내용이 넘쳤거나(잘림), 헤더 높이가 0이라 제목이 안 보임 등

---

## 8. 오른쪽 아래 리모컨 (저장·되돌리기)

| 버튼 | 기능 | 단축키 |
|---|---|---|
| ↩ **실행취소** | 직전 작업 취소 | Ctrl + Z |
| ↪ **다시실행** | 취소한 작업 복구 | Ctrl + Y |
| ⬇ **이미지 다운로드** (파란색) | 완성 이미지를 **JPG 파일**로 저장 | — |
| 💾 **백업 저장** (빨간색) | 작업 내용을 **파일(.json)** 로 저장 | Ctrl + S |
| ⬆ **백업 불러오기** | 저장한 백업 파일 다시 열기 | — |
| ⌃ **맨 위로 / 맨 아래로** | 항목 목록 이동 | — |

---

## 9. 저장과 백업 (중요!)

- 작업 내용은 **브라우저에 자동 임시 저장**되지만, **탭(창)을 닫으면 사라질 수 있습니다.**
- 그래서 **중요한 작업은 꼭 `백업 저장`(빨간 버튼)** 으로 파일을 받아 두세요.
  - 저장 파일 이름 예: `A4이벤트_백업_20260601_1430.json`
- 다음에 이어서 작업할 땐 **`백업 불러오기`** 로 그 파일을 열면 됩니다.

> ✅ **습관 추천:** 완성하면 ① 이미지 다운로드(인쇄용 JPG) + ② 백업 저장(.json) **둘 다** 받아 두기.

---

## 10. 자동 안전장치

- **의료법 금지어 감지** — 항목명에 광고 금지어(예: "신데렐라주사")를 입력하면 **경고와 함께 대체어**(예: "티옥트산주사")를 알려줍니다.
- **넘침 경고** — 내용이 한 장을 넘으면 미리보기 위에 경고가 떠 알려줍니다.

---

## 11. 자주 묻는 질문

**Q. 만들기 버튼이 없어요.**
→ 입력하면 자동으로 그려집니다. 별도 버튼이 없는 게 정상입니다.

**Q. 글자가 갑자기 커지거나 작아져요.**
→ `한 장에 자동 맞춤`이 켜져 있어 한 장에 맞추는 중입니다. 직접 정하려면 체크를 끄세요.

**Q. 작업하던 게 사라졌어요.**
→ 탭을 닫으면 임시 저장이 지워질 수 있습니다. 항상 `백업 저장`으로 파일을 받아 두세요.

**Q. 글자가 잘려요 / "다 안 들어간다"고 떠요.**
→ 항목이 너무 많습니다. 항목을 줄이거나, `우측 이벤트` 탭을 눌러 좌우 2단으로 나누세요.

**Q. 인터넷이 안 되면?**
→ 이미지 만들기·1장 다운로드는 됩니다. 단, 글꼴 모양·백업 저장은 인터넷이 필요할 수 있습니다.
